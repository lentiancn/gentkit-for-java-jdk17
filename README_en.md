## Introduction

Build a set of open-source frameworks with minimal dependencies and clean design, supporting independent usage of each
component, file, or even code segment without coupling between them.

## Development Language

- Java JDK 17, but if you need to use the old version environment, please proceed
  to [Java JDK 1.8](https://gitee.com/gentkit/gentkit-for-java-jdk8)
